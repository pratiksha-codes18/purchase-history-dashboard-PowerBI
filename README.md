## 📌 Overview
This project is an interactive Power BI dashboard designed to analyze purchase history data. It helps track purchasing trends, supplier performance, and overall cost analysis.

---

## 🎯 Key Features
- 📈 Total Purchase Cost Analysis
- 📦 Quantity Purchased Tracking
- 🏭 Supplier Performance Insights
- 📊 Category-wise Purchase Distribution
- 💳 Payment Method Analysis
- 📅 Monthly Purchase Trends

---

## 🛠️ Tools & Technologies
- Power BI
- Data Visualization
- Data Cleaning
- DAX (Data Analysis Expressions)

---

## 📂 Dataset Details
The dataset includes:
- Purchase ID
- Product ID
- Supplier Name
- Purchase Date
- Quantity
- Total Cost
- Payment Method
- Category

---

## 📊 Key Insights
- Identified top suppliers based on purchase volume
- Analyzed seasonal purchase trends
- Compared category-wise spending
- Tracked payment method usage

---

## 🚀 How to Use
1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Interact with filters and visuals
